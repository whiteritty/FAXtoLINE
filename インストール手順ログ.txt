こちらのリンクからget-pip.pyファイルをダウンロードしてください。

 https://bootstrap.pypa.io/get-pip.py?form=MG0AV3

ダウンロードフォルダに保存されていることを確認します。

ダウンロードしたファイルのディレクトリに移動する:

PowerShellで、get-pip.pyファイルが保存されているディレクトリに移動します。以下は一般的なコマンドの例です：

sh
cd C:\Users\ユーザー名\Downloads
get-pip.pyを実行する:

ダウンロードしたディレクトリに移動したら、以下のコマンドを実行します：

sh
python get-pip.py
pipコマンドを実行する:

get-pip.pyが正常に実行された後、以下のコマンドを再度試してください：

sh
pip install watchdog requests

これで解決できるはずです。もし問題が続く場合は、具体的なエラーメッセージを教えていただければ、さらに詳しくサポートできます。頑張ってください！
